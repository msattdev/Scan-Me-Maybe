# Scan-Me-Maybe

A simple QR code generator built with Python.

## Features

- Generate QR codes from text or URLs
- Save QR codes as PNG images
- Fast and lightweight

## Installation

1. Clone this repository:
```bash
git clone https://github.com/yourusername/Scan-Me-Maybe.git
cd Scan-Me-Maybe
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the scanner:
```bash
python3 scanmemaybe.py
```

This will generate a QR code image (e.g., `qrcode.png`) that you can scan with any QR code reader.
